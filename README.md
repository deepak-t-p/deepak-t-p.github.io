## ğŸ“ Customization

### Colors
Edit CSS variables in `assets/css/main.css`:
```css
:root {
    --primary-color: #3B82F6;
    --secondary-color: #8B5CF6;
    /* ... more variables */
}
```

### Content
Update content directly in HTML files:
- Personal information in sidebar
- Experience timeline in resume.html
- Projects in portfolio.html
- Contact details in contact.html

### Animation Speed
Adjust animation speed in `assets/css/main.css`:
```css
:root {
    --transition-fast: 0.2s ease;
    --transition-base: 0.3s ease;
    --transition-slow: 0.5s ease;
}
```
