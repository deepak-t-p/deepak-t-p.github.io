Transformer-Based Time Series Classification

Python PyTorch Transformers Time Series FFT Deep Learning

Overview

Developed a Transformer-based supervised learning model for patient disease classification using time-series temperature data. The dataset comprises 1,440 minute-level observations recorded over a 24-hour period, capturing granular physiological patterns that are indicative of various disease states.

Methodology

Exploratory Data Analysis (EDA)

Performed extensive exploratory data analysis using statistical techniques and informative visualizations to understand the underlying structure of the temperature time-series data. This phase was critical in uncovering temporal patterns, identifying anomalies, and informing feature engineering decisions.

Frequency Domain Analysis

Applied Fast Fourier Transform (FFT)–based frequency domain analysis to decompose the time-series signals into their constituent frequency components. This enabled the identification of periodic patterns and dominant frequencies that correlate with specific disease conditions, significantly improving feature understanding and downstream model performance.

Transformer Architecture

Designed and implemented a Transformer-based architecture tailored for sequential time-series classification. The model leverages self-attention mechanisms to capture long-range temporal dependencies across the 1,440 time steps, outperforming traditional RNN/LSTM baselines in both accuracy and training efficiency.

Key Features

  • Supervised classification of patient diseases from minute-level temperature readings
  • Comprehensive EDA with statistical analysis and rich visualizations
  • FFT-based frequency domain feature extraction for temporal pattern discovery
  • Custom Transformer encoder architecture optimized for time-series inputs
  • End-to-end pipeline from raw data processing to model evaluation

Technical Stack

  • Language: Python
  • Deep Learning: PyTorch, Transformer architecture
  • Data Analysis: Pandas, NumPy, SciPy (FFT)
  • Visualization: Matplotlib, Seaborn

Results

The Transformer model successfully captured complex temporal dependencies in the temperature data, enabling accurate multi-class disease classification. The FFT-based feature engineering provided complementary frequency-domain insights that improved model performance over purely time-domain approaches.
